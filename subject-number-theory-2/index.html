一、常系数一阶线性递推

递推式：ai=kai−1+b(k≠0)a_i=ka_{i-1}+b(k \neq 0)ai​=kai−1​+b(k​=0).

通项公式：an={a0+nbk=1a0kn+bk−1kn−bk−1k≠1a_n=\begin{cases}a_0+nb&k=1\\a_0k^n+\frac{b}{k-1}k^n-\frac{b}{k-1}&k\neq 1\end{cases}an​={a0​+nba0​kn+k−1b​kn−k−1b​​k=1k​=1​.

证明：构造 ai+x0=k(ai−1+x0)a_i+x_0=k(a_{i-1}+x_0)ai​+x0​=k(ai−1​+x0​) 后运用等比数列相关知识即可.

【例题】DTOJ 1378 随机数生成器

【题解】

直接分类讨论即可. 解方程时用 BSGS 求解.

【代码】

#include<bits/stdc++.h>
inline int read ( void )
{

inline int power ( int x,int y,int mod )
{
	int z=1;
	for ( ;y;y>>=1,x=1LL*x*x%mod ) if ( y&1 ) z=1LL*z*x%mod;
	return z;
}
std::unordered_map<int,int> map;
inline int BSGS ( int y,int z,int p )
{
	int m=(int)sqrt(p)+1;map.clear();
	if ( z%p==1 ) return 1;
	for ( int i=0,t=z;i<m;i++,t=1LL*t*y%p ) map[t]=i;
	for ( int i=1,v=power(y,m,p),t=v;i<=m+1;i++,t=1LL*t*v%p ) if ( map.count(t) ) return (i*m-map[t]+1)%p;
	return -1;
}
signed main()
{
	for ( int T=read();T--; )
	{
		int p=read(),a=read(),b=read(),X1=read(),t=read(),ans=-1;
		if ( X1==t ) ans=1;
		else if ( a==1 )
		{
			if ( b ) ans=(1LL*(t-X1+p)*power(b,p-2,p)+1)%p;
		}
		else if ( a==0 )
		{
			if ( b==t ) ans=2;
		}
		else
		{
			int res=1LL*b*power((a-1+p)%p,p-2,p)%p;
			ans=BSGS(a,1LL*(t+res)%p*power((X1+res)%p,p-2,p)%p,p);
		}
		if ( !ans ) ans+=p;
		printf("%d\n",ans);
	}
	return 0;
}

二、常系数二阶齐次线性递推

递推式：ai=pai−1+qai−2(p,q≠0)a_i=pa_{i-1}+qa_{i-2}(p,q \neq 0)ai​=pai−1​+qai−2​(p,q​=0).

通项公式：设 α,β\alpha,\betaα,β 为方程 x2−px−q=0x^2-px-q=0x2−px−q=0 两根，则 an=Aαn+Bβna_n=A\alpha^n+B\beta^nan​=Aαn+Bβn. 其中 A,BA,BA,B 为常数，可用待定系数法求解.

证明：构造 an−αan−1=β(an−1−αan−2)a_n - \alpha a_{n-1}= \beta(a_{n-1}-\alpha a_{n-2})an​−αan−1​=β(an−1​−αan−2​) 即可.

注意到 α,β\alpha,\betaα,β 常为无理数，可用二次剩余或矩阵维护.

【例题】DTOJ 4558「JLOI2015」有意义的字符串

【题解】

考虑共轭. 构造方程 x2+d−b24x+b=0x^2+\frac{d-b^2}{4}x+b=0x2+4d−b2​x+b=0. 设两根为 p,qp,qp,q. 则题目要求为 pnp^npn.

考虑 fn=pn+qn=(p+q)(pn−1+qn−1)−pq(pn−2+qn−2)=(p+q)fn−1−pqfn−2f_n=p^n+q^n=(p+q)(p^{n-1}+q^{n-1})-pq(p^{n-2}+q^{n-2})=(p+q)f_{n-1}-pqf_{n-2}fn​=pn+qn=(p+q)(pn−1+qn−1)−pq(pn−2+qn−2)=(p+q)fn−1​−pqfn−2​.

直接计算即可. 又 qnq^nqn 较小，对其讨论即可.

三、Fibonacci 循环节

显然 Fibonacci 数列在  mod p\bmod pmodp 意义下有循环节，且循环节长度不大于 p2p^2p2.

由于证明较为复杂，此处仅给出结论：

设 G(p)G(p)G(p) 为  mod p\bmod pmodp 意义下循环节. 对 ppp 分解有： p=∏i=1kpiaip=\prod\limits_{i=1}^{k} p_i^{a_i}p=i=1∏k​piai​​.

则 G(p)=∏i=1kG(piai)=∏i=1kG(pi)×piai−1G(p)=\prod\limits_{i=1}^{k} G(p_i^{a_i})=\prod\limits_{i=1}^{k} G(p_i) \times p_i^{a_i-1}G(p)=i=1∏k​G(piai​​)=i=1∏k​G(pi​)×piai​−1​.

对于 G(pi)G(p_i)G(pi​) 又有：若 pi2≡1(mod5)p_i^2 \equiv 1 \pmod 5pi2​≡1(mod5)，则 G(pi)∣p−1G(p_i) \mid p-1G(pi​)∣p−1. 否则 G(pi)∣2(p+1)G(p_i) \mid 2(p+1)G(pi​)∣2(p+1).

又实际上无需确切算出最小循环节，用 p−1p-1p−1 及 2(p+1)2(p+1)2(p+1) 代替即可.
